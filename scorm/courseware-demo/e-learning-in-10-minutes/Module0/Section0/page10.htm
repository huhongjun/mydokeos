This page shows you how to process photos for use in your e-learning course.

Suitable photos can usually be found from your company's marketing department or on the web at sites such as Flickr, Microsoft and Getty Images. Alternatively you can always take your own.

In this case you will be shown how to replace the image on this page.

The basic principle applies to all templates.

First we switch to Edit Mode and then highlight the image which is to be changed.

We also will need to make a note of the image size we need to replace on a piece of paper. In this case we can see it is 386 wide and 290 high.

We can now press the free Online Image Editor web application which will be used to resize our image.

NB You can use whatever application you like to resize the image (Photoshop, Fireworks etc.) however the basic principles remain the same.

First we open the image that is to be used. It is important that the image is not too large, ideally it should be less than 1600 pixels wide.

If the image is too large and/or your speed is too slow this can cause the application to time out.. If you need help with this contact UK 00 44 115 993 4230 or US 00 1 (646) 461-6210.

The image is uploaded.

You will see that it is much larger than needed so we will first resize it so that it can be cropped effectively.

The resize button is pressed and the image is resized to 500 pixels.

The height of the image resizes automatically to maintain the dimensions of the image (the aspect ratio.)

These changes are now applied to the image so they wont be lost when we proceed to crop the image.
Now we press the crop button and enter the width and height of the area we wish to crop (in this case it is 386 wide and 290 high.)
This cropped area can be moved over the centre of the image.

Apply the cropping and then save the image locally on your PC.

The image is now ready to be replaced

Press the replace image button and load in the newly cropped image.

The photo has now been replaced.
