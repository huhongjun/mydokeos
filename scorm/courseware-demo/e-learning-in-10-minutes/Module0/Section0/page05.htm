Page Level Changes

To edit the first default page in the course, navigate to that page using the Course Structure menu.

NB: e-Learning courses are built up from over 50 different templates available to you.  The page you will see before you is the most basic.

To make changes to this template you will need to switch to Edit Mode. 

This mode allows you to both use the e-learning (click on buttons, links etc.) but also edit the content at the same time.

The first step is to change the text.

Click on the passage of text so that your mouse pointer turns into a cursor.

The text will become highlighted enabling you to type directly over the content.

You will now be shown how to edit the image …

… click on the image so that it is highlighted. Press the Insert Image button   and load in an appropriate replacement (load in the file of the right size.)  You will see that the image has now changed.

You are now ready to add a new more interactive page to the course.
