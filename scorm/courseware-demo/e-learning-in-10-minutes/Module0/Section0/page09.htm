Questions

The e-CMS tool allows you to define 2 types of question: Test and Assessment.

Once the course is imported responses can be monitored using the EMS - Employee Management System).

Exam questions allow you to create tests with associated pass marks.

The system can be set up so that a test must be passed prior to proceeding with an assessment etc.

In order to add a Test question to the course, Navigate to the Test folder and select Insert Question.

Then select the question template.

Enter the question and potential responses with an associated score and feedback for the user.

These scores will be used to calculate pass marks etc.

NB: Some fields will be disabled since these apply to Assessment Questions.

Save the question and you will see how it will be displayed to the trainee.

Questions can be sorted, renamed and edited in the same way as normal training pages.

Assessment questions allow you to check aspects of your employees or their work environment which requires management intervention.

Assessment questions are typically used for auditing or carrying out employee checklists.

The question is defined as before except each response can have an associated task.

i.e. In this case the respondent is asked whether they have received the leaflet; if they answer "No" then this initiates a task  "Leaflet required".

Assessment questions are delivered to the trainee at the end of the e-learning session.

Any tasks created by the users responses are then monitored by the Employee Management System.

The system feeds each task to the appropriate personnel and or department.

The workflow is recorded, providing a complete audit trail of the process.

This is important additional functionality which lies outside the SCORM e-learning standard. 

As such these questions should only be used with the included EMS system - do not include these questions if working with a 3rd party LMS.
