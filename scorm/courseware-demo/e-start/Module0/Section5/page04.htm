If possible employ solutions on the left of the screen, selecting the next category along only if a satisfactory solution cannot be found.
Eliminate- Remove the hazard altogether

This is obviously the best and most effective control measure.

If this is not possible then move to the next Control Measure

Reduce/Replace

Can you reduce or replace the task/duty/operation with something that is safer Obviously this is not as effective as ‘Eliminate’ but nonetheless is second best.

From this point on any control measure that is put in place accepts the fact that the original hazard remains.

Isolate

Can the hazard be isolated?

E.g. could a noisy piece of equipment be put into a room of its own thereby ‘isolating it’ or could a group of potentially dangerous individuals be isolated from the main group?

Control

Can you control the hazard e.g. by using a safe system of work or can training be given to assist in controlling the hazard.

In the case of dangerous machinery can interlocked guards be installed.

Protection 

All too often managers opt to go straight for this option without following the ‘hierarchy’.

Protection should never be the first option and if it is used then sound reasoning needs to be identified so that, if necessary in the event of an accident, any investigating officer (possibly HSE Inspector) can see that the hierarchy of controls process has been followed.

Discipline

This measure is available in the event of non-compliance by individuals of either H&S Regulations or your organisation's own H&S rules.

Remember if a job cannot be done safely, it should not be done at all.
