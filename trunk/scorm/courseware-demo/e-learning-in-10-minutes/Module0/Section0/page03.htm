Starting up

If you are planning to edit an existing course first make a local copy of the selected course found in the "ecourses" folder on the  webserver. 

 

In this case we will create a new course.

Open the system by clicking on the e-CMS icon on your desktop.

e-CMS is divided into 4 sections.

1 Top toolbar

2 The course structure

3 Screen Content

4 Bottom toolbar

Move your mouse over the screen to learn more.

To create a new course…

Select the File menu and select New Course. This will create a new course with a  single page in a location of your choosing.

Pressing OK will create a new course in this location.

An empty course and structure are loaded into the system.

The first task is to make the course look and feel as if it belongs to your organisation. We call these Course Level changes...
