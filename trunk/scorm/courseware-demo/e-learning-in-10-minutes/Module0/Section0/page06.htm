Editing an Interactive Page

You will now add a new, more interactive, page to the course …

… click on the Insert Page  button at the bottom of the screen.

This is a list of all templates available to you.

Select the "branching" folder and from within this folder select the first template. 

This is the templated page which you will add to your course.

Press OK and the page is now added.

NB: The format of the page automatically updates to match the one chosen.

To edit the text on each option, once more switch to Edit Mode.

Then click on Option A so it is highlighted and then replace the text with your own.

Now click on the text in the bottom right hand corner of the page and replace the text. 

You can repeat these steps for each link entering whatever labels and text descriptions you chose.

 

Images can also be replaced in the same way as for basic pages.

 

There are many additional interactive template categories that can be used and edited in the same way.

For a full list click on the Insert Page button or consult the manual.
