Editing an Interactive Flash Page

Although no coding skills are necessary you will need Adobe/Macromedia Flash to customise these screens.

IF YOU DO NOT HAVE A COPY OF ADOBE FLASH SKIP TO THE NEXT SECTION

Click on the Insert Page   button at the bottom of the screen and open the "Flash" folder. 

From within this folder select the first template "1_panel_2_steps" and press OK. 

The template is now loaded into the Page Content.

The swf contained within the template shows you how to make customizations to the associated fla file which is in the flash subfolder. 

When you export the Flash file export it as a version 5 file.

Flash template categories that can should be edited in this way can be found in the Flash and Multiple Choice template folders.

You should now know how to add e-learning pages. We will now move on to sorting these pages into an e-learning course.
